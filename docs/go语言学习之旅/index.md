# Go语言学习之旅


## 前言

最近对Go语言非常感兴趣，准备花一段时间来学习，此系列文章用于学习笔记整理与学习记录。

## 简介

**Go**（又称 **Golang**）是 Google的 Robert Griesemer，Rob Pike 及 Ken Thompson 开发的一种静态强类型、编译型语言。Go 语言语法与 C相近，但功能上有：内存安全，GC（垃圾回收），结构形态及 CSP-style 并发计算。
