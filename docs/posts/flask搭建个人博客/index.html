本教程中我们将从0开始搭建一个完整的个人博客。

开发环境说明

本教程开发环境为MacOS Mojave10.14.6，Python版本为3.6.6，flask版本为1.0.2。

为了避免不必要的麻烦，尽可能的与我的开发环境保持一致。Python版本必须为3.x以上，建议3.6.x的版本，flask版本必须为1.0.x。
