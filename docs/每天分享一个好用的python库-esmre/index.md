# 每天分享一个好用的Python库-esmre

## 前言
今天分享是一个用于加速大型正则集合执行的第三方库：`esmre`。

## esmre

### 简介

当你需要从一大堆数据中找寻自己想要的数据时可以使用[AC自动机算法](https://zh.wikipedia.org/wiki/AC%E8%87%AA%E5%8A%A8%E6%9C%BA%E7%AE%97%E6%B3%95)使用AC自动机算法，在`Python`中这时候可以使用`esmre`。

### 安装

```python
pip install esmre
```

>>> index = esm.Index()
>>> index.enter('保罗')

>>> index.enter('贝弗利')
